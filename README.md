# Python Mini Projects Collection
This repository contains three beginner-level Python projects that help in learning basic programming concepts such as lists, loops, mathematical operations, random modules, and string manipulation.
These projects are useful for beginners to understand how real-world applications are built using simple logic in Python.

--------------------------------------------------

# Project 1: To-Do List

## Goal
Build a program where users can add tasks such as "Finish Python assignment" to a list and view them later.

### Key Skills
- Lists
- Append method
- Loops
- Basic user input

### Why This Project Matters
This project teaches how to store multiple values inside a single variable using lists. This concept is very important because it is the foundation for databases and data storage systems.

### Example Tasks
- Study Python
- Finish assignment
- Submit project

--------------------------------------------------

## Project 2: Expense Tracker

### Goal
Create a program where users enter expense amounts such as 100, 50, or 20 and the program calculates the total money spent.

### Key Skills
- Mathematical operations
- Variables
- Accumulator logic
- User input handling

### Why This Project Matters
This project teaches how programs process numbers and calculate totals, which is very important in financial applications and backend systems.

### Example
Expense 1: 100  
Expense 2: 50  
Expense 3: 20  

Total Spent: 170

--------------------------------------------------

## Project 3: Random Password Generator

### Goal
Create a tool that asks the user for the password length (for example 8 characters) and generates a strong random password using letters and numbers.

Example Password:
P@ssword_G3n3r@t0r!

### Key Skills
- Importing modules
- Using Python random library
- String manipulation

### Why This Project Matters
This project helps developers understand how to use Python's built-in libraries to create secure and random passwords, which is useful in real-world security systems.
